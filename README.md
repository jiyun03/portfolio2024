# 폴더 구조

```
project/
|-- public/                 // 정적 파일 (아이콘 SVG를 포함한 이미지, favicon.ico 등)
|-- src
|-- |-- components/         // 컴포넌트
|-- |-- context/            // 상태관리
|-- |-- pages/              // 페이지 컴포넌트
|-- |-- theme/              // 테마 설정
|-- |-- types/              // 타입 설정
|-- |-- utils/              // 유틸리티 및 도우미 함수
|-- next.config.mjs         // next 설정 파일
|-- package.json            // 프로젝트 정보 및 종속성 정의
|-- tsconfig.json           // typescript 설정 파일
```

# 환경 및 사용 라이브러리

- React.js
- Next.js
- Typescript
- styled-components
- react-spring
- tua-body-scroll-lock
- Three.js
